# Tech Territory Map Pro v3.0

Render-ready technician coverage and territory management app.

## Deploy
1. Upload this folder to GitHub.
2. Create a Render Web Service.
3. Build command: `npm install`
4. Start command: `npm start`

## Features
- Draw/edit/delete technician territories
- Drag polygon points using Leaflet Draw edit mode
- Multi-tech priority list per territory
- Search by ZIP/city/state/area/address
- State sidebar for NY, NJ, DE, PA, GA
- Add/edit/delete techs, states, ZIPs, cities, notes
- Auto-zoom to working areas
- Import/export/backup/restore JSON/CSV
- Responsive desktop/mobile UI

Data is saved in browser localStorage for simple deployment.
